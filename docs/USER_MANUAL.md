# 用户操作手册

> IOTsys 物联网充电桩管理平台 · 用户操作指南 v1.0

---

## 一、系统概述

IOTsys 是一套面向充电桩运营商的物联网管理平台，支持电单车充电桩（AP3000 协议）和电动汽车充电桩（TF100 协议）的统一接入、监控与管理。

### 1.1 适用角色

| 角色        | 权限范围                                       |
| ----------- | ---------------------------------------------- |
| **管理员**   | 全部功能：设备管理、站点管理、指令下发、告警处理、用户管理、系统配置 |
| **运维人员** | 设备监控、告警查看与处理、指令下发（只读配置）     |
| **查看者**   | 仪表盘、设备列表、报表查看（只读）                |

### 1.2 访问地址

| 环境     | 地址                    |
| -------- | ----------------------- |
| 开发环境 | `http://localhost:3000` |
| 生产环境 | `https://iot.your-domain.com` |

### 1.3 登录

1. 打开系统地址，进入登录页面
2. 输入用户名和密码
3. 首次登录建议修改默认密码

> 默认管理员账号由系统初始化时创建，请联系系统管理员获取。

## 三、设备管理

路径：侧边栏 → **设备管理**

### 3.1 设备列表

设备列表展示所有已注册设备，支持：

- **筛选**：按设备类型（电单车桩 / 汽车桩）、协议、状态、站点
- **搜索**：按设备 SN 或安装位置模糊搜索
- **排序**：按创建时间、状态排序
- **分页**：默认每页 20 条

列表字段说明：

| 字段       | 说明                                              |
| ---------- | ------------------------------------------------- |
| 设备 SN    | 设备唯一序列号                                     |
| 设备类型   | 电单车充电桩 / 电动汽车充电桩                       |
| 协议       | AP3000_v2 / TF100_v1                              |
| 厂商/型号  | 设备厂商和型号                                     |
| 安装位置   | 站点 + 具体位置                                    |
| 状态       | 🟢 在线 / ⚫ 离线 / 🔴 故障 / 🟡 维护中              |
| 固件版本   | 当前设备固件版本                                   |
| 最后上线   | 最近一次 TCP 连接时间                               |

### 3.2 添加设备

1. 点击"添加设备"按钮
2. 填写设备信息：

| 字段           | 必填 | 说明                            |
| -------------- | ---- | ------------------------------- |
| 设备 SN        | ✅   | 设备序列号，与设备端配置一致      |
| 设备类型       | ✅   | 电单车充电桩 / 电动汽车充电桩    |
| 协议           | ✅   | AP3000_v2 或 TF100_v1           |
| 厂商           |      | 如：XX 科技                      |
| 型号           |      | 如：AP3000-10S                  |
| 所属站点       |      | 从站点列表中选择                 |
| 安装位置       |      | 如：B1 层 001 车位               |
| 固件版本       |      | 初始固件版本号                   |

3. 点击"确定"保存

> **注意**：设备 SN 必须与设备端配置的 SN 完全一致，否则设备无法正常注册上线。

### 3.3 设备详情

点击设备列表中的设备 SN 或"详情"按钮，进入设备详情页。

详情页包含：

#### 实时数据面板

| 数据项       | 说明                     |
| ------------ | ------------------------ |
| 电压 (V)     | 设备当前输入电压          |
| 电流 (A)     | 设备当前总电流            |
| 功率 (W)     | 设备当前总功率            |
| 累计电量     | 设备历史累计充电量 (kWh)  |
| 今日电量     | 当日充电量 (kWh)         |
| 温度 (℃)     | 设备内部温度              |
| 信号强度     | 网络信号 RSSI             |
| 充电状态     | 空闲 / 充电中 / 已充满 / 故障 |

#### 端口/充电枪状态

**电单车桩**（AP3000）：
- 每个端口显示：端口号、状态（空闲/充电中/已刷卡/短路/浮充/继电器粘连）、电压、电流、功率

**汽车桩**（TF100）：
- 每个充电枪显示：枪号、状态（空闲/已连接/充电中/已完成/预约/故障）、电压、电流、功率、SOC、充电时长

#### 指令历史

展示该设备最近下发的指令列表，包含指令类型、状态、下发时间、响应时间。

### 3.4 下发指令

在设备详情页，点击"下发指令"按钮：

#### 电单车桩（AP3000）指令

| 指令类型     | 说明                 | 参数                |
| ------------ | -------------------- | ------------------- |
| 远程启动     | 启动指定端口充电      | 端口号、充电时长(分) |
| 远程停止     | 停止指定端口充电      | 端口号              |
| 查询状态     | 查询设备全部端口状态  | 无                  |
| 参数配置     | 配置设备运行参数      | 参数名 + 值         |
| 远程重启     | 重启设备             | 无                  |
| 清除存储     | 清除设备本地存储      | 无                  |

#### 汽车桩（TF100）指令

| 指令类型     | 说明                 | 参数                |
| ------------ | -------------------- | ------------------- |
| 远程启动     | 启动充电枪充电        | 枪号                |
| 远程停止     | 停止充电枪充电        | 枪号                |
| 费率下发     | 下发计费费率          | 费率配置 JSON       |
| 费率查询     | 查询当前费率          | 无                  |
| 参数配置     | 配置设备参数          | 参数名 + 值         |
| 远程重启     | 重启设备             | 无                  |
| 地锁控制     | 控制地锁升降          | 地锁地址、动作      |
| 语音下发     | 下发语音播报内容      | 语音内容            |

指令状态说明：

| 状态      | 含义           |
| --------- | -------------- |
| pending   | 等待下发       |
| sent      | 已发送，等待响应 |
| responded | 设备已响应     |
| success   | 执行成功       |
| failed    | 执行失败       |
| timeout   | 响应超时       |

### 3.5 编辑 / 删除设备

- **编辑**：修改站点、安装位置、状态（维护中标记）
- **删除**：仅可删除离线且无进行中订单的设备（需二次确认）

## 五、告警管理

路径：侧边栏 → **告警管理**

### 5.1 告警列表

展示所有告警记录，支持按状态（活跃/已处理/已确认）、严重级别、设备筛选。

告警类型：

| 告警类型   | 说明                 | 严重级别 |
| ---------- | -------------------- | -------- |
| 设备离线   | 设备心跳超时          | ⚠️ 警告   |
| 设备故障   | 设备上报故障码        | 🔴 严重   |
| 过载       | 功率超出额定值        | 🔴 严重   |
| 高温       | 温度超出安全阈值      | ⚠️ 警告   |
| 漏电       | 漏电保护触发          | 🔴 严重   |
| 继电器粘连 | 继电器无法断开        | 🔴 严重   |
| 急停       | 紧急停止按钮被按下     | 🔴 严重   |

### 5.2 处理告警

1. 点击告警记录进入详情
2. 查看告警详情（设备信息、发生时间、故障描述）
3. 执行处理操作：
   - **确认**：标记为已知悉
   - **处理**：执行修复后标记为已解决，填写处理备注
4. 处理完成后告警自动归档

## 九、常见操作场景

### 场景 1：新设备上线

1. 在"设备管理"页面点击"添加设备"
2. 填写设备 SN（与设备端配置一致）、类型、协议
3. 设备上电联网后，自动连接平台 TCP 端口（7000）
4. 平台自动识别协议并建立会话
5. 返回设备列表，确认设备状态变为"在线"

### 场景 2：远程启动充电

1. 在设备列表找到目标设备，点击进入详情
2. 确认设备状态为"在线"
3. 点击"下发指令"
4. 选择指令类型：远程启动
5. 填写端口号 / 枪号
6. （可选）设置充电时长限制
7. 点击"确认下发"
8. 在指令历史中查看执行结果

### 场景 3：处理设备故障告警

1. 仪表盘或告警管理页面出现新告警
2. 点击告警进入详情，查看故障描述和故障码
3. 根据故障码对照协议文档排查问题：
   - 继电器粘连 → 需现场检修
   - 漏电保护 → 检查线路绝缘
   - 高温 → 检查散热风扇
4. 现场处理完成后，在告警详情页点击"已解决"
5. 填写处理备注

### 场景 4：批量查看站点运营情况

1. 进入"数据报表"页面
2. 选择时间范围（如：本月）
3. 查看"站点充电排名"报表
4. 导出 Excel 做进一步分析

## 十、常见问题

**Q: 设备 SN 填写错误怎么办？**
A: 可在设备管理页面编辑设备信息，修改 SN。但需确保设备端也同步修改。

**Q: 设备一直显示离线？**
A: 检查设备是否上电联网、TCP 端口 7000 是否可达、设备 SN 是否与平台一致。

**Q: 指令下发后一直显示 pending？**
A: 设备可能已离线。检查设备在线状态，确认网络通畅。

**Q: 如何批量导入设备？**
A: 当前版本支持单条录入，批量导入功能在后续版本规划中。
